# 💱 Currency Converter App

A responsive and interactive web application that allows users to convert between multiple currencies in real-time. This project uses a currency exchange API to fetch live rates and provides a clean, user-friendly interface.

---

## 📌 Overview

Currency conversion is a common real-world problem. This app simulates a practical tool for users to quickly convert currencies using live exchange rates. It demonstrates front-end development with **JavaScript API integration**, responsive design, and usability.

---

## 🚀 Features

* 🔄 Convert between multiple currencies
* 🌐 Real-time exchange rates using API
* 🧾 Input validation for amounts
* 📱 Fully responsive design for desktop, tablet, and mobile
* 🎨 Clean UI with HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

* **HTML** – Structure of the app
* **CSS** – Styling and responsive layout
* **JavaScript** – Logic and API integration
* **API** – Real-time currency exchange data (e.g., ExchangeRate API or open API)

## 🧠 How It Works

1. User selects **from-currency** and **to-currency**
2. User enters an **amount**
3. JavaScript fetches **real-time exchange rate** from API
4. Converted amount is displayed instantly
