# 💱 Currency Converter App

A real-time currency converter web application that allows users to convert between different currencies using live exchange rates from an API.

---

## 🚀 Features

* Real-time currency conversion
* API integration for live exchange rates
* User-friendly interface
* Responsive design
---

## 🛠️ Tech Stack

* HTML
* CSS

* Exchange Rate API

## ⚙️ How It Works

1. User selects base currency
2. User selects target currency
3. Enters amount
4. App fetches exchange rate via API
5. Displays converted result instantly
